# Shore Trace 公考学习记录

手机优先、本地优先的公考学习记录工具，目标是围绕一个轻量闭环长期使用：

快速记录 -> 自动保存 -> 留下明日重点 -> 第二天继续执行

## 当前版本

当前版本已完成记录闭环、历史查看、历史详情、历史编辑入口、JSON 导出和统计概览能力。

已支持：

- 选择记录日期
- 自定义日期选择面板
- 历史日期补录
- 动态选择当天学习过的模块
- 记录学习时长、做题数、错题数
- 固定错因标签
- 自定义错因
- 模块备注
- 今日收获
- 明日重点
- 自动保存和手动保存
- 本地数据校验与异常保护
- 历史记录列表
- 历史记录详情
- 历史记录编辑入口
- JSON 数据导出
- 统计页

暂未实现：

- PWA 离线安装
- 数据导入
- 登录、后端、云同步

## 技术栈

- Vue 3
- Vite
- Tailwind CSS
- Vue Router
- localStorage

## 本地开发

```bash
npm install
npm run dev
```

手机预览时，让手机和电脑连接同一个 Wi-Fi，然后运行：

```bash
npm run dev -- --host 0.0.0.0
```

用手机浏览器打开终端显示的局域网地址，并访问 `/record`。

## 测试与构建

```bash
npm test
npm run build
```

## 数据说明

数据仅保存在当前浏览器的 `localStorage` 中。清理浏览器数据、切换浏览器或切换设备后，当前版本不会自动同步数据。

当前版本支持在历史页导出 JSON 数据，尚未支持数据导入或多设备同步。`localStorage` 容量有限，也可能被浏览器清理；如果检测到本地数据结构异常，应用会优先阻止继续覆盖保存，以避免把可能可恢复的数据静默清掉。
